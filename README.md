# 🌌 AstroLinux

Planificador de observación astronómica hecho con **Qt6 + Qt Quick**.

### Características
- Gestión de múltiples ubicaciones
- Pronóstico del tiempo nocturno (Open-Meteo)
- Fase lunar
- Objetos recomendados para observar
- Modo oscuro optimizado para observación

### Tecnologías
- Qt 6.5+
- QML
- CMake
- C++

---

**Hecho por un sysadmin Linux y aficionado a la astronomía.**
